# Certificate-Autofiller-System

Certificate-Autofiller-System is a Jupyter-based automation tool designed to streamline the generation and distribution of personalized certificates. It eliminates the manual effort of placing names on templates and managing cloud permissions by automating the entire pipeline from data ingestion to shareable link generation. 

This repository contains a high-throughput automation engine specifically engineered to handle the certification needs for thousands of participants at the RXCU2026 Open House (Faculty of Pharmaceutical Sciences, Chulalongkorn University).
